## ğŸ“– Usage Guide

### Adding a New Task

1. Click the **"+ Add New Task"** button
2. Enter a **heading** for your task
3. Add a detailed **description**
4. Click **"Save"** to add it to your daily log

### Managing Tasks

- **Edit**: Click the edit icon on any task card
- **Delete**: Click the delete icon to remove a task
- **Reorder**: Drag and drop tasks to reorganize
- **Search**: Use the search bar to filter tasks

### Sharing Your Work

1. Click the **"Share"** button in the header
2. Copy the generated shareable link
3. Send it to anyone - they can view your daily work log

### Export/Import Data

- **Export**: Click "Export Data" to download your tasks as JSON
- **Import**: Click "Import Data" to restore from a backup file
